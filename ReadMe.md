## About the project

This repository contains an example project that combines WDIO, Mocha, and TypeScript to facilitate efficient testing for your mobile applications. This README provides essential information on setting up, running tests, and maintaining the project.

<!-- GETTING STARTED -->

## Getting started

### Prerequisites

Before running the tests, make sure you have the following dependencies installed:

Node.js: Install Node.js from https://nodejs.org/en

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/nikdesu/Mobile-Automation-on-Browserstack---WebdriverIO-Task-8
   ```
2. Install the project dependencies:
   ```bash
   npm install
   ```
   <p align="right">(<a href="#readme-top">back to top</a>)</p>
<!-- USAGE EXAMPLES -->
## Running test

To run the test suite, execute the following command:

```bash
npm run pixel3a.ci
```
OR
```bash
npm run pixel7pro.ci
```
